> 约定：全程 SI 单位。每个 shift **恰好**破坏 *baseline* 的一条对称律 / 守恒律；所列其它不变量均经验证保留（解析 + 待 invariant-checker 数值复核）。随机分布为 *目录级占位* — 最终范围在 scenario 标定时确定。

# Part A: 域 1–6（力学 + 电磁）

域选理由：D6 v1 偏重力学覆盖。我们挑选 NewtonBench 中六个域，经调研验证的 shift 能够干净地隔离单一对称性，同时数值表现温和：**Hooke 弹簧、Newton 引力、阻尼谐振子、单摆、Coulomb 力、RLC 电路**。它们合计覆盖 `LIN, PAR, ROT, SCALE/Bertrand, T-trans, T-rev, Q-守恒, Onsager-互易` 等 8 条不同的对称性破缺，分布在 18 个 shift 中。

## Domain 1: Hooke 弹簧（1D & 2D）

**Baseline 律**：`F = −k x`。维度：`[F]=kg·m·s⁻²`, `[k]=kg·s⁻²`, `[x]=m`。
**不变量**：T-trans（E 守恒）、PAR（`x→−x`）、T-rev、LIN、无内禀长度（scale-free），2D 中额外 ROT。

### Tier-1 (γ) — 结构 / 对称性破缺

- **shift-γ-1-1 — 饱和非对称刚度**
  - 动机：键合式的拉伸-压缩非对称。
  - 破缺：PAR（`x→−x`）。
  - 修改后律：`F(x) = −k x · [1 + η tanh(x/x₀)]`。
  - 维度：`[η]=1, [x₀]=m`。输入 `x [m]` → 输出 `F [N]`。
  - 保留：T-trans（自治 → E 存在，势 `V(x)=∫₀ˣ k s(1+η tanh(s/x₀)) ds` 良定义且下有界）、T-rev（无 `ẋ`）、小幅极限下的 LIN（`|x|≪x₀`）。
  - 采样：`k ~ LogUniform(1, 100) N/m`；`η ~ Uniform(0.1, 0.8)`；`x₀ ~ LogUniform(0.05, 2.0) m`。
  - 安全范围：仿真 `|x| ≤ 4 x₀`（势阱保持单极小，无逃逸）。
  - 文献角度（未复制）：非对称 Morse 型势阱 — 我们将指数尾换成有界 `tanh`，使 `F` 在大 `|x|` 时保持线性。

- **shift-γ-1-2 — 2D 各向异性刚度（双折射的跨域 re-skin）**
  - 动机：将单轴双折射（光学）re-skin 到力学。
  - 破缺：ROT（平面各向同性）。
  - 修改后律：力由势 `V(r, θ) = ½ K(θ) r²` 导出，`K(θ) = k₀ [1 + ξ cos(2(θ − φ))]`，`F = −∇V` ⇒
    - 径向分量 `F_r = −K(θ) r`，
    - 切向分量 `F_θ = −(1/r) ∂V/∂θ = k₀ ξ r · sin(2(θ − φ))`。
    切向分量是 `L_z` 不守恒的结构来源（破缺 ROT 所对应的 Noether 荷）。
  - 维度：`[k₀]=kg·s⁻²`、`[ξ]=1`、`[φ]=rad`。输入 `(x,y) [m]` → `F [N]`。
  - 保留：T-trans（E 存在因 `V` 与时间无关且 `F = −∇V` 保守）、T-rev（无 `ẋ`）、PAR（`r→−r ⇒ θ→θ+π`，`cos(2θ+2π)=cos(2θ)`、`sin(2θ+2π)=sin(2θ)`）、LIN、关于原点的 S-trans。
  - 角动量 `L_z` **不**保留（自洽：ROT 是唯一破缺的对称性，`L_z` 是其 Noether 荷）。
  - 采样：`k₀ ~ LogUniform(1, 100) N/m`；`ξ ~ Uniform(0.1, 0.7)`；`φ ~ Uniform(0, π)`。
  - 安全：`ξ < 1` 保证所有方向上刚度均为正。

### Tier-2 (δ) — 守恒律破缺

- **shift-δ-1-1 — 幅度调制的阻尼（Stokes drag 的跨域 re-skin）**
  - 动机：将粘性阻尼 re-skin 到 Hooke 玩具中，使耗散受幅度门控。
  - 破缺：**能量守恒**（T-trans *保留* — 自治 — 但沿轨迹 `dE/dt < 0`，源于速度耦合力项；这是显式耗散嵌入力律的标准 "T-trans 完好、E 不守恒" 模式）。
  - 修改后律：`F = −k x − c (x²/L²) ẋ`。
  - 维度：`[c]=kg/s`、`[L]=m`。输入 `(x, ẋ)` → `F [N]`。
  - 保留：T-trans（自治）、PAR（在 `x→−x, ẋ→−ẋ` 下：`F → −F`）、1D 各向同性。
  - T-rev：作为耗散的标准结果而丢失（计入能量守恒破缺的一部分 — 无其它独立对称性受影响）。
  - 采样：`k ~ LogUniform(1, 100) N/m`；`c ~ LogUniform(1e-3, 1) kg/s`；`L ~ LogUniform(0.5, 5) m`。
  - 安全：`c x_max² / (L² · √(km)) ≤ 0.3` 使系统保持次临界阻尼。
  - 文献角度：非线性阻尼族（平方律拖曳），但我们附加了 `x²/L²` 包络使耗散在平衡位置附近消失 — 无公开的规范名称。

## Domain 2: Newton 引力（两体）

**Baseline 律**：`F = −G m₁ m₂ r̂ / r²`。维度：`[G]=m³·kg⁻¹·s⁻²`。
**不变量**：T-trans（E）、S-trans（p）、ROT（L）、T-rev、PAR、GAL、束缚轨道的 Bertrand 闭合性、等效原理。

### Tier-1 (γ)

- **shift-γ-2-1 — 四极各向异性耦合**
  - 动机：将各向异性材料弹性 re-skin 到引力耦合，且零单极偏移，使远场测试看起来标准。
  - 破缺：ROT。
  - 修改后律：由势 `V(r, r̂; n̂) = −G₀ m₁ m₂ · [1 + ξ ((r̂·n̂)² − ⅓)] / r` 导出，力 `F = −∇V`。显式分量（设 `μ ≡ r̂·n̂`）：
    - 径向 `F_r = −G₀ m₁ m₂ [1 + ξ(μ² − ⅓)] / r²`，
    - 切向 `F_⊥ = +(2 G₀ m₁ m₂ ξ μ / r²) · (n̂ − μ r̂)`。
    切向分量提供使 `L` 守恒破缺的力矩（与 ROT 的 Noether 配对）。
  - 维度：`[G₀]=m³·kg⁻¹·s⁻²`、`[ξ]=1`，`n̂` 无量纲单位向量。输入 `(r̂, r [m])` → `F [N]`。
  - 保留：T-trans（E 守恒 — `V` 与时间无关且 `F = −∇V`）、S-trans、T-rev、PAR（`r→−r ⇒ μ→−μ`，`μ²` 偶，`n̂−μr̂ → −(n̂−μr̂)` 随 `r̂` 翻号 ⇒ `F` 作为真实矢量变换）、GAL。
  - L **不**守恒（唯一破缺的对称性是 ROT — L 是其 Noether 荷）。
  - 采样：`G₀ = G·LogUniform(0.5, 2)`；`ξ ~ Uniform(0.05, 0.4)`；`n̂ ~ Uniform on S²`。
  - 安全：`ξ < 0.5` 保证括号因子在所有方向均为正（`μ²−⅓ ∈ [−⅓, ⅔]` ⇒ 因子 ∈ `[1 − ξ/3, 1 + 2ξ/3] > 0`）。
  - 文献角度：Bianchi 各向异性 / SME 引力，但**无迹四极**结构 + 零单极去除了 Yukawa 式的全局重标度 — 函数形式独特。

- **shift-γ-2-2 — Lorentzian 量程凸起（破 Bertrand）**
  - 动机：引入内禀长度尺度扰动闭合轨道，但不改变远场 `1/r²`。
  - 破缺：SCALE 不变性（束缚轨道的 Bertrand 闭合性 → 轨道进动）。
  - 修改后律：`F = −G m₁m₂ /r² · [1 + α (r/r₀) / (1 + (r/r₀)²)]`。
  - 维度：`[α]=1`、`[r₀]=m`。输入 `r [m]` → `F [N]`。
  - 保留:ROT（仍为中心力 → L 守恒）、T-trans（E 存在）、T-rev、PAR、GAL、S-trans。
  - 采样：`α ~ Uniform(0.05, 0.5)`；`r₀ ~ LogUniform(0.1, 10) · r_typical`。
  - 安全：强制 `r ≥ 1e-3 · r₀` 以避免近碰撞奇异（baseline `1/r²` 中已存在，未加剧）。
  - 文献角度：第五力 / Yukawa 屏蔽；我们用尖峰 Lorentzian 包络替代常规的指数形式，并以 `1/r²` 上的乘性凸起出现，而非加性项。

### Tier-2 (δ)

- **shift-δ-2-1 — G 的慢谐波调制**
  - 动机：将参数泵浦（声学）re-skin 到引力常数。
  - 破缺：T-trans（E 不守恒）。
  - 修改后律：`F = −G(t) m₁m₂ r̂/r²`，`G(t) = G₀ [1 + β cos(ω_G t)]`（相位固定为 0，使 `G(t)` 关于 `t=0` 为偶）。
  - 维度：`[β]=1`、`[ω_G]=s⁻¹`。输入 `(r̂, r [m], t [s])` → `F [N]`。
  - 保留：ROT（中心力 ⇒ L 守恒）、S-trans（p 守恒）、PAR、T-rev（余弦关于 `t=0` 时间偶；仅 T-trans 是破缺的对称性）。
  - 采样：`G₀ = G·LogUniform(0.5, 2)`；`β ~ Uniform(0.05, 0.3)`；`ω_G ~ LogUniform(1e-4, 1e-2) · ω_orbit`。无随机相位。
  - 安全：`β·ω_G·T_sim ≤ 0.5` 使 `G` 在仿真窗口内的净漂移有界。
  - 文献角度：Dirac LNH（线性 `G(t)` 漂移）— 我们以*有界*正弦调制替换，使仿真不会发散。

## Domain 3: 阻尼谐振子

**Baseline 律**：`ẍ + 2γ ẋ + ω₀² x = 0`。维度：`[γ]=s⁻¹`、`[ω₀]=s⁻¹`。
**Baseline 的不变量**（含阻尼之后）：T-trans（自治）、PAR（`x→−x`）、LIN。（注：T-rev 与 E 已被 `2γẋ` 破坏 — 它们在本域中不是进一步 γ shift 的可用目标。）

### Tier-1 (γ)

- **shift-γ-3-1 — 幅度记忆刚度（非线性声学的跨域 re-skin）**
  - 动机：刚度缓慢追踪近期振幅 — 借自滞回颗粒介质。
  - 破缺：LIN。
  - 修改后律：`ẍ + 2γẋ + ω₀²·[1 + κ ⟨x²⟩_τ / x_ref²]·x = 0`，其中 `⟨x²⟩_τ = (1/τ)∫_{t−τ}^t x²(s) ds`。
  - 维度：`[κ]=1`、`[τ]=s`、`[x_ref]=m`。输入 `(x, ẋ, history)` → 运动方程。
  - 保留：T-trans（窗口随 `t` 滑动）、PAR（`x²` 为偶）。T-rev / E 在 baseline 中已缺失。
  - 采样：`ω₀ ~ LogUniform(0.5, 10) rad/s`；`γ/ω₀ ~ LogUniform(0.01, 0.3)`；`κ ~ Uniform(0.05, 0.5)`；`τ ~ Uniform(0.5, 5)/ω₀`；`x_ref ~ LogUniform(0.1, 2) m`。
  - 安全：`κ ≤ 0.6` 保证有效 `ω` 为实且有界。
  - 文献角度：Duffing 式刚度，但**幅度是有限窗口上的滑动均值**，而非瞬时值 — 与任何标准具名振子区分。

- **shift-γ-3-2 — 含阻尼的慢参数泵浦**
  - 动机：把弹簧泵浦得足够慢，使其位于主 Mathieu 舌之下。
  - 破缺：T-trans。
  - 修改后律：`ẍ + 2γẋ + ω₀²·[1 + ε cos(Ω_p t)] x = 0`。
  - 维度：`[ε]=1`、`[Ω_p]=s⁻¹`。
  - 保留：PAR、LIN。
  - 采样：`ε ~ Uniform(0.05, 0.3)`；`Ω_p ~ Uniform(0.3, 1.7) ω₀`（刻意*避开* 2ω₀ 与 ω₀ 以避免主 / 次参数共振）。
  - 安全：`ε < 4γ/ω₀`（参数不稳定性的次阈值）。

### Tier-2 (δ)

- **shift-δ-3-1 — 幅度门控的阻尼符号反转**
  - 动机：阻尼在"核心"区域内变为反阻尼，在区域外为耗散。
  - 破缺：单调能量耗散（此耗散 baseline 中可用的守恒律类比）。系统自发安顿到极限环上。
  - 修改后律：`ẍ + 2γ·(|x|/L − 1)·ẋ + ω₀² x = 0`。
  - 维度：`[L]=m`。输入 `(x, ẋ)` → 运动方程。
  - 保留：T-trans、PAR（`x→−x, ẋ→−ẋ` 保持方程不变）、刚度的 LIN。
  - 采样：`ω₀ ~ LogUniform(0.5, 10)`；`γ/ω₀ ~ LogUniform(0.01, 0.2)`；`L ~ LogUniform(0.1, 2) m`。
  - 安全：极限环半径 `~2L`；总机械能有界于 `2 k L²`。
  - 文献角度：van der Pol 提供了状态门控负阻尼的*想法*；我们用分段线性包络 `(|x|/L − 1)` 替代 `(1 − x²)` 多项式，使阈值几何在性质上不同。

## Domain 4: 单摆（平面、刚臂）

**Baseline 律**：`θ̈ + (g/L) sin θ = 0`。维度：`[g/L]=s⁻²`。
**不变量**：T-trans（E）、`θ→−θ`（PAR）、T-rev、关于竖直方向的反射对称。

### Tier-1 (γ)

- **shift-γ-4-1 — 非对称竖直方向（破 PAR）**
  - 动机：将"偏置转子"势（分子物理）re-skin 到平面单摆 — 在恢复力矩中加入 θ 偶分量，使运动方程失去 `θ→−θ` 不变性。
  - 破缺：PAR（`θ→−θ`）。
  - 修改后律：`θ̈ + (g/L)·sin θ + (g/L)·α·(1 − cos θ) = 0`。
  - 维度：`[α]=1`。输入 `θ [rad]` → `θ̈ [rad·s⁻²]`。
  - 保留：T-trans（自治 → E 存在，势 `V(θ) = (g/L)(1−cos θ) + (g/L) α (θ − sin θ)`，两项在 `θ=0` 附近均良定义且下有界）、T-rev（无 `θ̇`）。`θ=0` 处稳定平衡保留：`V'(0)=0`，`V''(0)=g/L > 0`。
  - PAR 校验：`(1−cos θ)` 偶，`sin θ` 奇 ⇒ 在 `θ→−θ` 下，`θ̈` 翻号但 `(g/L) α (1−cos θ)` 项不变 ⇒ EOM 不再不变 ⇒ PAR 确实破缺。
  - 采样：`g/L ~ LogUniform(1, 100) s⁻²`；`α ~ Uniform(0.05, 0.5)`。
  - 安全：`α < 1` 保持 `θ=0` 处局部线性化稳定，并防止在摆动窗口 `|θ| ≤ π/2` 内出现次级平衡。
  - 文献角度：Josephson 动力学的倾斜搓板势提供非对称摆势的*想法*；我们使用 `α(1−cos θ)` 而非常数倾斜 `−αθ`，使系统保持周期性且偏置在平衡处消失。

- **shift-γ-4-2 — 高度分级的引力（破 S-trans）**
  - 动机：从天体力学借用潮汐梯度，re-skin 到桌面单摆。
  - 破缺：竖直 S-trans（沿引力轴的平移不变性）。
  - 修改后律：`θ̈ + (g₀/L)·[1 − α·(L(1−cos θ))/H]·sin θ = 0`。
  - 维度：`[α]=1`、`[H]=m`。
  - 保留：T-trans（E 存在；势 `V(θ) = m g₀ L (1−cos θ) − m g₀ α L²/(2H)(1−cos θ)²`）、PAR（`θ→−θ` 保持 `(1−cos θ)` 不变）、T-rev。
  - 采样：`g₀/L ~ LogUniform(1, 100) s⁻²`；`α ~ Uniform(0.02, 0.3)`；`H ~ LogUniform(0.5, 50) · L`。
  - 安全：`α·L/H < 0.5` 使有效重力在摆动范围内保持正。

### Tier-2 (δ)

- **shift-δ-4-1 — 离共振的引力调制**
  - 动机：参数驱动频率刻意选在参数不稳定舌*之外*。
  - 破缺：T-trans（E 不守恒）。
  - 修改后律：`θ̈ + (g(t)/L)·sin θ = 0`，`g(t) = g₀·[1 + ε cos(Ω t)]`。
  - 维度：`[ε]=1`、`[Ω]=s⁻¹`。
  - 保留：PAR、T-rev（余弦关于 `t=0` 偶；随机 `Ω` 下按本目录约定仅 T-trans 被破缺 — `φ` 固定为 0）。
  - 采样：`g₀/L ~ LogUniform(1, 100) s⁻²`；`ε ~ Uniform(0.05, 0.3)`；`Ω ~ Uniform(0.3, 1.7) ω₀`（避开 `ω₀` 与 `2ω₀` 邻域）。
  - 安全：`ε ≤ 0.4 · |Ω/(2ω₀) − 1|`（主 Mathieu 舌的次阈值）。
  - 文献角度：竖直枢轴驱动（Kapitza 式），但属于慢速离共振泵浦而非快速稳定区 — 动力学区间不同。

## Domain 5: Coulomb 力（静电、多电荷）

**Baseline 律**：`F_{ij} = k q_i q_j r̂_{ij}/r_{ij}²`；总 `F_i = Σ_j F_{ij}`。
维度：`[k]=N·m²·C⁻²`、`[q]=C`。
**不变量**：T-trans（E）、S-trans（p）、ROT（L）、T-rev、PAR、Q（总电荷）、U(1)、叠加（LIN）。

### Tier-1 (γ)

- **shift-γ-5-1 — 各向异性 Coulomb 耦合**
  - 动机：将光学的单轴介质各向异性 re-skin 到裸静电力，从势导出以保力的保守性。
  - 破缺：ROT。
  - 修改后律：成对势 `V_pair(r, r̂; m̂) = (k q_i q_j / r) · [1 + χ ((r̂·m̂)² − ⅓)]`，力 `F_{ij} = −∇_{x_i} V_pair`。显式分解（设 `ν ≡ r̂_{ij}·m̂`）：
    - 径向 `F_{ij,r} = (k q_i q_j / r_{ij}²) · [1 + χ(ν² − ⅓)]`，
    - 切向 `F_{ij,⊥} = −(2 k q_i q_j χ ν / r_{ij}²) · (m̂ − ν r̂_{ij})`。
    电荷 `i` 上的总力：`F_i = Σ_{j≠i} F_{ij}`（成对可加）。
  - 维度：`[χ]=1`，`m̂` 无量纲单位向量。
  - 保留：T-trans（E 守恒 — `V_pair` 与时间无关且保守）、S-trans（p 守恒）、T-rev（无 `v`）、PAR（反演下为矢量）、Q、LIN / 叠加（仍成对可加）。
  - L 不保留（与 ROT 单一 Noether 配对的损失）。
  - 采样：`k = k_Coulomb·LogUniform(0.5, 2)`；`χ ~ Uniform(0.05, 0.4)`；`m̂ ~ Uniform on S²`。
  - 安全：`χ < 0.5` ⇒ 括号因子在所有方向为正（`ν²−⅓ ∈ [−⅓, ⅔]`）。

- **shift-γ-5-2 — 饱和势非线性（非线性弹性的跨域 re-skin）**
  - 动机：从非线性弹簧借用"软"立方饱和，应用在*势*层面，使得到的场仍保持无旋（能量守恒），而源到场的映射为非线性。
  - 破缺：LIN（场的叠加）。
  - 修改后律：从源集定义线性标量势 `φ_lin(x) = Σ_j k q_j / |x − x_j|`。有效势 `φ_eff(x) = φ_lin + ξ · φ_lin³ / (φ_lin² + φ₀²)`。试验电荷上的场：`E_eff = −∇φ_eff`；试验电荷 `q_t` 上的力：`F = q_t E_eff`。
  - 维度：`[ξ]=1`、`[φ₀]=V`。`[φ_eff]=V` ✓。
  - 保留：T-trans（E 守恒 — `φ_eff` 与时间无关 ⇒ `E_eff = −∇φ_eff` 保守）、ROT（映射 `φ_lin → φ_eff` 是标量场的标量函数 ⇒ 旋转等变）、PAR（在反演下，对位置取反的电荷集 `φ_lin` 不变；场作为矢量变换）、Q、T-rev。
  - 采样：`k = k_Coulomb·LogUniform(0.5, 2)`；`ξ ~ Uniform(0.05, 0.5)`；`φ₀ ~ LogUniform(0.1, 10) · φ_typical V`。
  - 安全：`ξ < 1` 保持 `φ_eff(φ_lin)` 在 `φ_lin` 上单调（导数 `1 + ξ · φ_lin²(φ_lin² + 3 φ₀²)/(φ_lin² + φ₀²)² > 0`）。
  - 文献角度：Born–Infeld 非线性电动力学启发了饱和 EM 响应的*想法*；我们将非线性置于势层面（不同的数学层次）并使用有理立方饱和形式而非 Born–Infeld 平方根。

### Tier-2 (δ)

- **shift-δ-5-1 — 场耦合电荷泄漏**
  - 动机：电荷以由局部场设定的速率缓慢漏入周围"真空" — 将力学阻尼 re-skin 到电荷变量本身。
  - 破缺：Q 守恒。（T-rev 在静态 baseline 中是平凡的，作为同包的一部分丢失；按 D6 约定计为单条守恒律破缺。）
  - 修改后律：`dq_i/dt = −α · (|E_loc,i|/E_ref)^n · q_i`，其中 `E_loc,i = Σ_{j≠i} k q_j (x_i − x_j)/|x_i − x_j|³`。力律本身不变。
  - 维度：`[α]=s⁻¹`、`[n]=1`、`[E_ref]=V/m`。
  - 保留：T-trans（速率仅依赖瞬时状态）、S-trans、ROT、PAR。
  - 采样：`α ~ LogUniform(1e-4, 1e-1) s⁻¹`；`n ~ Uniform(0.5, 2)`；`E_ref ~ LogUniform(0.1, 10)·E_typical V/m`。
  - 安全：`α · T_sim ≤ 1` 使总电荷漂移为亚 1 阶。
  - 文献角度：毫电荷 / Okun 式电荷不守恒，但使其*确定且场门控*，而非自发。

## Domain 6: RLC 电路（单回路或双耦合回路）

**Baseline 律（单回路）**：`L · di/dt + R · i + q/C = V_src(t)`，`i = dq/dt`。维度：`[L]=H`、`[R]=Ω`、`[C]=F`、`[q]=C`、`[i]=A`、`[V]=V`。
**不变量**：T-trans（能量预算 `½ L i² + ½ q²/C + ∫R i² dt = const`）、LIN、`q↔−q` 宇称（无偏置回路中）、多回路情形下的 Onsager 互易性。

### Tier-1 (γ)

- **shift-γ-6-1 — 可饱和电感（铁磁饱和的跨域 re-skin）**

  - 破缺：LIN。

  - 维度：`[L₀]=H`、`[I_sat]=A`。
  - 保留：T-trans（若 `V_src` 自治）、`q↔−q` 宇称（`L(i)=L(−i)`）、Onsager（单回路，平凡成立）。

  - 安全：仿真 `|i| ≤ 5 I_sat`（电感不会趋于零）。
  - 文献角度：铁磁芯饱和；我们使用平滑的 Lorentzian `1/(1+u²)` 而非教科书的 `tanh` 形 B–H 曲线。

- **shift-γ-6-2 — 非互易互感（双回路）**
  - 动机：借用 gyrator 式非互易，但实现为非对称互感 `M_{12} ≠ M_{21}`，而非理想集中 gyrator 元件。
  - 破缺：Onsager 互易性（即多回路线性网络的 T-rev）。
  - 修改后律（两回路 i=1,2）：
    `L_i di_i/dt + Σ_{j≠i} M_{ij} di_j/dt + R_i i_i + q_i/C_i = V_i(t)`，`M_{12} = M₀ + δM/2`，`M_{21} = M₀ − δM/2`，`δM ≠ 0`。

  - 保留：T-trans（自治部分）、LIN（仍对 `i, q` 线性）、`q↔−q` 宇称、每个回路上的 Q。
  - 注：E 非独立守恒；该损失与 T-rev/Onsager 破缺 Noether 配对，按 Part A 耗散打包惯例计为单条破缺（与 δ-1-1、δ-3-1、δ-6-1 同款）。
  - 采样：`L_i ~ LogUniform(1e-3, 1) H`；`R_i ~ LogUniform(0.1, 100) Ω`；`C_i ~ LogUniform(1e-9, 1e-5) F`；`M₀ ~ Uniform(0, 0.5)·√(L₁ L₂)`；`δM/M₀ ~ Uniform(0.05, 0.4)`。
  - 安全：`|M₀ ± δM/2| < √(L₁ L₂)`（电感矩阵正定，无发散）。
  - 文献角度：超材料中的非互易磁耦合；我们让元件保持纯电感（无理想 gyrator），因此机制是非对称的 `M`。

### Tier-2 (δ)

- **shift-δ-6-1 — 参数电感调制**

## Domain 7：热传导（Fourier 律）

**Baseline law**：`q_i = −k ∂_i T`，等价 `∂_t T = α ∇² T`（`α = k/(ρ c_p)`）。
量纲：`[T]=K, [q]=W·m⁻², [k]=W·m⁻¹·K⁻¹, [α]=m²·s⁻¹`。
**Invariants**：SO(3) 各向同性、S-trans、T-trans、`T→T+const`（baseline 只用 `∇T`）、能量积分 `∫ρc_p T dV` 在绝热边界守恒、Onsager 对称 `k_{ij}=k_{ji}`、抛物自相似 scale `(t→λt, x→λ^{1/2} x)`。

### Tier-1 (γ) —— 结构性 / 对称性破坏

- **shift-γ-7-1 —— 常数 β 各向异性热导（介电张量的跨域重皮）** *(round-1 修复 B1)*
  - 动机：借电磁各向异性介电张量 `ε_{ij}=ε₀(δ_{ij}+χ n_i n_j)` 的代数形式搬到热域。原 `tanh(|∇T|/G₀)` 门控被 invariant-checker 指出会*独立*破抛物 self-similar scale，造成 dual break；改用**常数 β + 大宽对数随机 + 每 scenario 随机 `n̂`**，仅破 SO(3)。
  - 破坏：SO(3)（出现优先方向 `n̂`）。
  - 修改律：`q_i = −K_{ij} ∂_j T`，`K_{ij} = k₀ [δ_{ij} + β · n_i n_j]`。
  - 量纲：`[k₀]=W·m⁻¹·K⁻¹`，`[β]=1`，`n̂` 为无量纲单位向量。输入 `T [K], ∇T [K/m]` → `q [W/m²]`。
  - 保留：S-trans，T-trans，`T→T+c`（K 与 T 无关），能量守恒（K 对称正定 ⇒ 散度形式），Onsager（K_{ij}=K_{ji}），抛物自相似 scale `(t→λt, x→λ^{1/2}x)` ✓（K 与场无关 ⇒ scale 不变）。
  - 采样：`k₀ ~ LogUniform(0.1, 50) W·m⁻¹·K⁻¹`；`β ~ LogUniform(0.05, 5)`（两 decade 宽分布以保 non-textbook 强度）；`n̂ ~ Uniform on S²`。
  - 安全范围：`β > -1 ⇒ K` 严格正定；上限 `β < 5` 保平行方向传导 ≤ 6×正交 —— 数值良态。
  - 文献角度：借介电张量代数结构；`β` 大宽对数随机 + 各 scenario 独立 `n̂` ⇒ 区别于 textbook 单晶各向异性 Fourier（系数窄、方向固定）。

- **shift-γ-7-2 —— 幂律记忆核（分数阶粘弹性的跨域重皮）**
  - 动机：复杂介质（聚合物、多孔）有时间记忆；避开经典 Cattaneo–Vernotte 双曲方程，借分数阶力学的核形式。
  - 破坏：抛物自相似 scale `(t→λt, x→λ^{1/2}x)`（核引入幂律时间标度 `p`，scale 变为 `x→λ^{(1-p)/2} x` —— 与 baseline 不同）。
  - 修改律：`q_i(t) = − ∫_{t₀}^{t} G(t − s) · ∂_i T(s) ds`，`G(τ) = k₀ · τ^{-p} / Γ(1 − p)` for `τ ≥ τ_min`，`τ < τ_min` 用 `τ_min^{-p}/Γ(1-p)` 截断。
  - 量纲：`[G]=W·m⁻¹·K⁻¹·s^{p−1}`；`[q]=W/m²` ✓。
  - 保留：时间平移（卷积形式 + `t₀ → -∞` 极限取大窗口近似），SO(3)，S-trans，`T→T+c`，能量守恒（核重写为分数 Fick 律仍是散度律），Onsager。
  - 采样：`p ~ Uniform(0.10, 0.55)`；`k₀ ~ LogUniform(0.1, 50)`；`τ_min = Δt_sim`（不送给 AI）。
  - 安全范围：`p < 0.6` 保 `Γ(1-p)` 良态且积分收敛；下截断防 `τ=0` 奇异。
  - 文献角度：借 fractional heat conduction 框架，但 `p` 自由对数随机、不锁定 Caputo / Riemann-Liouville 具体定义 —— 与文献固定阶数版本区分。

### Tier-2 (δ) —— 守恒律破坏

- **shift-δ-7-1 —— 带共动参考的过剩量平方汇（quadratic-in-excess sink with comoving reference）** *(round-1 修复 B2)*

  - 破坏：能量守恒（`∫ρc_p T dV` 不再守恒；T-rev 按 Part A 约定 bundled）。

  - 量纲：`[λ]=s⁻¹`，`[T_ref]=K`，`[⟨T⟩]=K`；RHS `[K/s]` ✓。
  - 保留：SO(3)，S-trans，T-trans（自治），`T→T+c` ✓（`T→T+c ⇒ ⟨T⟩→⟨T⟩+c ⇒ (T−⟨T⟩) 不变 ⇒` sink 项不变），Onsager（扩散部分）。
  - 采样：`λ ~ LogUniform(1e-5, 1e-2) s⁻¹`；`T_ref ~ LogUniform(50, 1000) K`；`α` 同 baseline。
  - 安全范围：关于 `⟨T⟩` 的二次中心吸引 ⇒ T 围绕动态均值收敛；初值 `T(0) ≥ 2 · max|T(0) − ⟨T(0)⟩|` 保证 `T ≥ 0` 全程；不发散。
  - 文献角度：借辐射冷却思想，弃 `T⁴` 标准式；共动二次形式非 textbook（标准 Newton's cooling 用绝对参考）。

## Domain 8：标量波（1D / 2D 声学，无损 baseline）

**Baseline law**：`∂_t² u = c² ∂_x² u` (1D)，2D：`∂_t² u = c² ∇² u`。量纲：`[u]=m, [c]=m·s⁻¹`。
**Invariants**：T-trans，S-trans，parity `x→−x`，T-rev，Lorentz-like w/ `c`，能量 `E = ∫(½(∂_t u)² + ½ c²|∇u|²)dx` 守恒，线性叠加，SO(2/3) 各向同性 (≥2D)。

### Tier-1 (γ)

- **shift-γ-8-1 —— 三阶手性色散（KdV-linear motif 的跨域重皮）**
  - 动机：借 KdV 线性色散 `∂_x³` 算子搬到 d'Alembert 方程作声波 —— 文献无此组合。
  - 破坏：parity `x→−x`（`∂_x³` 奇）。
  - 修改律：`∂_t² u = c² ∂_x² u + γ · c² · ∂_x³ u`。
  - 量纲：取 `[γ]=m`；则 `[γ c² ∂_x³ u] = m · m²·s⁻² · m·m⁻³ = m·s⁻²` ✓。
  - 保留：T-trans，S-trans，T-rev（项含 `∂_t²` 偶，`∂_x³ u` 时间偶 ⇒ `t→-t` 保留），能量守恒（色散项纳入 Hamilton 形式 `H = ∫(½π² + ½c²(∂_x u)² + ½γc²·(...))`，色散非耗散），线性叠加。
  - 采样：`γ ~ Uniform(-L₀, L₀)`，`L₀ ~ LogUniform(1e-4, 1e-1) m`（双符号）；`c ~ LogUniform(50, 5000) m/s`。
  - 安全范围：色散非耗散 ⇒ 振幅不增；CFL 需 `Δt ≤ Δx² / (c · |γ|_max)`（高阶差分）。
  - 文献角度：KdV linear part 搬到声学；与 textbook acoustic dispersion (`∂_x⁴` Boussinesq-like) 区分。

- **shift-γ-8-2 —— 2D 各向异性相速度（介电张量的跨域重皮）**
  - 动机：晶格 / 多层介质各向异性 —— 借电磁介电张量代数到声学。
  - 破坏：SO(2) 平面旋转。
  - 修改律：`∂_t² u = c² ∂_i (M_{ij} ∂_j u)`，`M = R(θ₀) · diag(1, 1+β) · R(θ₀)ᵀ`（主轴系下 diag）。
  - 量纲：`[M]=1`；RHS `[m/s²]` ✓。
  - 保留：T-trans，S-trans，parity（`x→-x`：`M` 在主轴系仍对称），T-rev，能量守恒（M 对称正定 ⇒ Hamilton 良态）。
  - 采样：`β ~ Uniform(0.1, 0.8)`；`θ₀ ~ Uniform(0, π)`；`c ~ LogUniform(50, 5000) m/s`。
  - 安全范围：`β < 1 ⇒ M` 正定 ⇒ 双曲，标准 CFL 适用。
  - 文献角度：介电张量代数到声学；与 textbook anisotropic elasticity 张量（4 阶）维度不同 —— 2 阶张量是 cross-domain 简化。

### Tier-2 (δ)

- **shift-δ-8-1 —— 振幅门控粘性阻尼（耗散单标签）** *(round-1 修复 B3)*
  - 动机：原版 `−α₀ ∂_t u` 是 textbook 线性阻尼波动方程，lookup-style AI 一眼识别 —— invariant-checker 标记 attack-resistance fail；采用**振幅门控**子线性包络（option a），与 Part A δ-3-1 同款 motif 跨域到 PDE。
  - 破坏：能量守恒（与 Part A δ-1-1 / δ-3-1 / δ-6-1 同款"dissipative ⇒ E + T-rev = 单条"约定；非线性状态依赖阻尼按 dissipative bundle 处理）。
  - 修改律：`∂_t² u = c² ∂_x² u − α₀ · (|u| / u_ref) · ∂_t u`。
  - 量纲：`[α₀]=s⁻¹`，`[u_ref]=m`；`[α₀ · |u|/u_ref · ∂_t u] = s⁻¹ · 1 · m/s = m/s²` ✓。
  - 保留：T-trans（自治），S-trans（系数 x-无关），parity `x→-x` ✓（项不动）；`u→-u` 下 `|u|→|u|, ∂_t u→-∂_t u, ∂_t² u→-∂_t² u` ✓。
  - 采样：`α₀ ~ LogUniform(1e-3, 0.3) s⁻¹`；`u_ref ~ LogUniform(1e-3, 1) m`；`c ~ LogUniform(50, 5000) m/s`。
  - 安全范围：振幅小 ⇒ 阻尼 → 0（线性极限恢复 baseline），大振幅 ⇒ 强阻尼自限；振幅有界，不发散。
  - 文献角度：与 Part A δ-3-1（van der Pol-modified 振幅门控负阻尼）同源 motif 但作用在 PDE 而非 ODE，且为纯耗散 —— 与 textbook 线性吸收 `e^{-α t}` lookup 不匹配。

## Domain 9：几何光学 / Snell

**Baseline law**：`n₁ sin θ_1 = n₂ sin θ_2`。Fresnel 在标量近似下：`R + T = 1`。
量纲：`[n]=1, [θ]=rad`。
**Invariants**：界面切向波矢守恒、互易性（光路可逆 / time-reversal of rays）、Fermat 平稳路径、偏振 U(1)（标量近似下 polarization 旋转不变）、能量 `R+T=1`、绕界面法线轴 SO(2)、介质交换 `1↔2` 形式对称。

### Tier-1 (γ)

- **shift-γ-9-1 —— 偏振调制折射率（晶体双折射 motif 的跨域重皮）**
  - 动机：借 birefringence 思想，但 `sin²(2θ_pol − φ)` 双瓣形（周期 π）非标准单轴形式。
  - 破坏：偏振 U(1)（标量近似下，baseline 对 polarization 角度旋转不变；该 shift 引入显式 polarization 依赖）。
  - 修改律：`n_eff(θ_pol) = n_0 + δn · sin²(2 θ_pol − φ)`；Snell：`n_1 sin θ_1 = n_eff(θ_pol) sin θ_2`。
  - 量纲：均无量纲 ✓。
  - 保留：互易性（固定 `θ_pol` 后路径可逆），Fermat，切向波矢守恒，SO(2) 绕法线，能量 `R+T=1`，介质 `1↔2` 对称（替换 `n_2 → n_eff` 而 `n_1` 独立）。
  - 采样：`n_0 ~ Uniform(1.3, 2.2)`；`δn ~ Uniform(0.02, 0.30)`；`φ ~ Uniform(0, π)`；`θ_pol ~ Uniform(0, π)` per ray。
  - 安全范围：`n_eff = n_0 + δn · [0,1] > 0` 总成立；TIR 临界角随 `θ_pol` 动态变化但有限。
  - 文献角度：借单轴 / 双轴晶体思想；`sin²(2θ-φ)` 双瓣 + 非介质张量直接形式 ⇒ counterfactual。

- **shift-γ-9-2 —— 介质交换非对称 Snell（非互易磁耦合 motif 的跨域重皮）**
  - 动机：借 Part A γ-6-2 非互易磁耦合思想搬到界面光学 —— 同样制造 `1↔2` 不对称。
  - 破坏：介质交换对称 / 互易性（光路时间反演不再完全对称）。
  - 修改律：`sin θ_t = (n_1/n_2) sin θ_i + κ · (n_1 − n_2)/(n_1 + n_2) · sin³ θ_i`。该 cubic 项在 `1↔2` 交换下变号 ⇒ 单破互易性。
  - 量纲：无量纲 ✓。
  - 保留：SO(2) 绕法线，Fermat（每个固定方向分支内仍平稳），能量 `R+T=1`（另独立建模），切向波矢守恒（在该方向上），偏振 U(1)。parity 在界面平面内的 `x→-x`（与法向无关）保留。
  - 采样：`n_1, n_2 ~ Uniform(1.0, 2.0)` 独立；`κ ~ Uniform(0, 0.15)`。
  - 安全范围：在 `κ < 0.2` 且 `|sin θ_i| < 0.95` 范围内 `|sin θ_t| ≤ 1`；超出按 TIR 处理（与 baseline 同款边界）。
  - 文献角度：手性 / 非互易光学思想，但 cubic-in-`sin θ` 项 + `(n_1-n_2)/(n_1+n_2)` 反对称因子组合非 textbook。

### Tier-2 (δ)

- **shift-δ-9-1 —— 角度幂律能量非平衡（Part A δ-5-1 场门控泄漏 motif 的跨域重皮）**
  - 动机：制造能量泄漏（或反向"获得"）—— 借 δ-5-1 中"law 不变，附加守恒律破"的思路，搬到光学界面。形式非 Beer-Lambert / Fresnel 标准。
  - 破坏：能量守恒（`R + T ≠ 1`）。
  - 修改律：角度方程保持 Snell baseline；强度方程 `R + T = 1 − ξ · |sin θ_i|^p`。允许 `ξ` 双符号。
  - 量纲：无量纲 ✓。
  - 保留：Snell 角度律本身，互易性（损失系数对入/出射方向同），SO(2) 绕法线，Fermat，切向波矢，偏振 U(1)，介质交换。

  - 文献角度：借吸收 / 倏逝耦合思想；幂律角度依赖 + 双符号随机系数 ⇒ counterfactual，非 Beer-Lambert / Fresnel。

## Domain 10：无粘流体（Bernoulli，稳态流线）

**Baseline law**：`½ ρ v² + ρ g h + p = const` 沿流线；附 `∇·v = 0`。
量纲：`[ρ]=kg·m⁻³, [v]=m/s, [h]=m, [p]=Pa, [g]=m·s⁻²`。
**Invariants**：质量守恒 `∇·v=0`，水平 Galilean，`h→h+const`（baseline 重力势吸收进 const），SO(3) 各向同性、时间平移（steady）、流线能量、可逆性（无粘）、SO(2) 水平旋转。

### Tier-1 (γ)

- **shift-γ-10-1 —— 各向异性动能惯性（有效质量张量的跨域重皮）** *(round-1 修复 B4 —— 补充 Euler 方程)*
  - 动机：取向性多相流（纤维悬浮 / 拉伸聚合物）—— 借固态有效质量张量代数到流体动能。Round-1 invariant-checker 指出 Bernoulli 是 Euler 的流线积分 ⇒ 必须给出对应的 momentum equation。
  - 破坏：SO(3) 各向同性（出现优先方向 `n̂`）。
  - 修改后动量方程（Euler 层）：`M_{ij} (∂_t v_j + (v·∇) v_j) = −∂_i p − ρ g ∂_i h`，`M_{ij} = ρ [δ_{ij} + α (n_i n_j − δ_{ij}/3)]`（traceless 各向异性 ⇒ `tr(M) = 3ρ` 不变）。
  - 修改后 Bernoulli（上 Euler 方程的流线积分）：`½ v_i M_{ij} v_j + ρ g h + p = const`（沿同一流线；由上 Euler 沿 `v` 缩并积分得出，`M` 对称 ⇒ KE-quadratic form 良定义）。
  - 量纲：`[M]=kg·m⁻³`；KE 项 `[Pa]` ✓；Euler 两侧 `[kg·m⁻²·s⁻²]` ✓。
  - 保留：质量守恒 `∇·v=0`，水平 Galilean（M 与 v 平移无关），`h→h+const`，T-trans（steady 项自治），流线能量（M 对称正定 ⇒ 沿 streamline 守恒），可逆性（无粘）。
  - 采样：`α ~ Uniform(-0.4, 1.4)`；`n̂ ~ Uniform on S²`；`ρ ~ LogUniform(50, 5e3) kg/m³`。
  - 安全范围：`α > -1 ⇒ M` 正定（最小本征 `ρ(1 − α/3) > 0` for `α<3`，最大 `ρ(1 + 2α/3)`，全正于 `α∈(-1, ∞)` 实用区间）。
  - 文献角度：借固态有效质量张量；搬到流体动能 + 完整 Euler 一致性 ⇒ counterfactual。与 Part A γ-1-2 / γ-5-1 / γ-7-1 是同源 anisotropy motif 的不同 instantiation。

- **shift-γ-10-2 —— 非线性重力势（分层大气 motif 的跨域重皮）** *(round-1 修复 B5 —— 采样约束修正)*
  - 动机：浮力分层 / 非均匀有效重力，避免 textbook 等温 / 等熵公式。Round-1 invariant-checker 指出原静态边界 `|h|<10·h₀` 在 `λ=-0.25, q=2` 下失败 (1+(-0.25)·100=-24) ⇒ 改为**采样级约束**。
  - 破坏：垂直平移 `h → h + const`（势能不再仿射于 `h`）。
  - 修改律：`½ ρ v² + ρ g h · (1 + λ (h/h_0)^q) + p = const`。
  - 量纲：势能项 `[Pa]` ✓。
  - 保留：水平 S-trans，SO(2) 水平旋转，水平 Galilean，`∇·v=0`，T-trans，流线能量，可逆性。
  - 采样预条件（sim 层在 scenario 发出前强制）：给 scenario 的物理 envelope `h_max`，按以下采样：`q ~ Uniform(0.5, 2.0)`；`h_0 ~ LogUniform(1, 100) m`；`ε(q, h_max, h_0) = 0.5 / (h_max/h_0)^q`；`λ ~ Uniform(−ε, min(ε, 0.5))`；拒绝 `|λ| < 0.01` 邻域避免 baseline 退化；`g = 9.81 m/s²`。
  - 安全范围：上述 precondition 保 `|λ|·(h_max/h_0)^q < 0.5 ⇒ (1 + λ(h/h_0)^q) ∈ (0.5, 1.5)` 在整个 `|h| ≤ h_max` 上严格正 ⇒ 势能单调凸 ⇒ 不发散。
  - 文献角度：借分层浮力；幂律高度修正 + `λ` 双符号 + 自适应采样约束 ⇒ counterfactual。与 Part A γ-4-2（tidal gradient pendulum）是同源 "non-uniform `g`" motif 的流体 instantiation。

### Tier-2 (δ)

- **shift-δ-10-1 —— 流线路径积分损耗（分布式电阻 motif 的跨域重皮）**
  - 动机：分布式 sub-grid 损失（多孔介质 / 弱湍流耗散），非 Darcy-Weisbach 标准形式。
  - 破坏：流线能量守恒。
  - 修改律：`½ ρ v² + ρ g h + p + ζ ∫_streamline |v − v_∞|^m ds = const`（沿同一流线）。`v_∞` 设为远场参考速度以保 Galilean。
  - 量纲：`[ζ] = kg · m^{-1-m} · s^{-m+2}`（按 `m` 调整使损耗项 `[Pa]`）；RHS `[Pa]` ✓。
  - 保留：`∇·v=0`，水平 Galilean（损失项写为 `|v − v_∞|^m` ⇒ frame-shift 保留），T-trans，SO(3) 各向同性，`h→h+const`，可逆性**失效** —— 按 Part A δ 同款"dissipative ⇒ 单一标签"惯例算一条。
  - 采样：`ζ ~ LogUniform(1e-4, 1e-1)`（单位随 `m` 调整）；`m ~ Uniform(1.5, 2.8)`；`v_∞ ~ Uniform(0, v_typical) m/s`（per scenario）。
  - 安全范围：积分单调增 ⇒ 仅能耗，总能上界 = inlet 能量；流场不发散。
  - 文献角度：借分布式 Darcy 思想，但路径积分 + 幂律相对速度 + `v_∞` 参考 frame ⇒ counterfactual。

## Domain 11：反应动力学（单步速率律）

**Baseline law**：`dC/dt = −k C^n`；配 Arrhenius `k = A exp(−E_a / (R T))`。
量纲：`[C] = mol·m⁻³, [k] = (mol·m⁻³)^{1−n}·s⁻¹, [E_a] = J·mol⁻¹, R = 8.314 J·(mol·K)⁻¹`。
**Invariants**：时间平移，Arrhenius 温度依赖，化学计量守恒（链 `A → B` 1:1 ⇒ `C_A + C_B = const`），正定性 `C ≥ 0`，dimensional homogeneity，稀释自相似 `C → λC`（baseline 在 `n=1` 时严格 scale-invariant，`n≠1` 时仅幂律自相似）。

### Tier-1 (γ)

- **shift-γ-11-1 —— 分数时间动力学（反常扩散 motif 的跨域重皮）**
  - 动机：受限介质（凝胶 / 多孔 / 反应-扩散耦合）下的非 Markov 反应；借 Domain 7 γ-7-2 同源 fractional kernel 但作用于不同 PDE。
  - 破坏：反应自相似 scale `(t→λt, C→λ^{−1/(n−1)}C)`（baseline 此 scale 律自洽；分数阶 `β` 改变 scale 指数）。
  - 修改律：`D_t^β C = −k C^n`（Caputo 分数时导）。
  - 量纲：`[D_t^β C] = mol·m⁻³·s^{-β}`；`k` 携相应 `s^{-β}` 单位 ⇒ ✓。
  - 保留：T-trans（卷积分数导对 `t→t+c` 不变，取 `t_0 → -∞` 极限近似），Arrhenius (`k = A e^{-E_a/RT}`)，正定性（`β,n` 合适区间），化学计量守恒（`dB = -dA` 同步分数化 ⇒ `C_A + C_B = const`），稀释 scale 仅在 `n=1` 极限。
  - 采样：`β ~ Uniform(0.55, 0.95)`；`n ~ Uniform(0.5, 2.5)`；`k ~ LogUniform(1e-4, 1e-1)`（单位随 `β,n` 自动调整）；历史窗口截断 `τ_min = Δt_sim`。
  - 安全范围：`β > 0.5` + `n ≥ 0.5` + 截断 ⇒ 稳定且 `C ≥ 0`。
  - 文献角度：借 anomalous reaction kinetics；`β` 自由对数随机 + Caputo / RL 不锁定 ⇒ counterfactual。

- **shift-γ-11-2 —— 浓度饱和速率（可饱和非线性 motif 的跨域重皮）**
  - 动机：拥挤 / 自抑制，避开 Michaelis-Menten 固定形式。借 Part A γ-5-2 / γ-6-1 "saturating Lorentzian" motif 到反应速率。
  - 破坏：稀释自相似 `C → λ C`（饱和引入内禀浓度尺度 `C_sat`）。
  - 修改律：`dC/dt = −k C^n / (1 + (C / C_sat)^m)`。
  - 量纲：分母无量纲；分子 `mol·m⁻³·s⁻¹` ⇒ ✓。
  - 保留：T-trans，Arrhenius，化学计量守恒，正定性，dimensional homogeneity。
  - 采样：`n ~ Uniform(1.0, 3.0)`；`m ~ Uniform(0.5, 3.0)`；`C_sat ~ LogUniform(1, 1e4) mol/m³`；`k ~ LogUniform(1e-4, 1e-1)`。
  - 安全范围：分母 > 0 always；高 `C` 渐近 `dC/dt ~ -k C^{n-m}`，要求 `n - m > -1` 以避免数值停滞（实现时验证 `m < n + 1`）。
  - 文献角度：借饱和动力学；`(n, m)` 独立随机 + 双幂律分母 ⇒ 与 Michaelis-Menten / Langmuir 固定 form 区分。

### Tier-2 (δ)

- **shift-δ-11-1 —— 隐通道分支损耗**
  - 动机：制造"总摩尔损耗到暗通道 / 热"的非标准副反应。
  - 破坏：化学计量守恒（`C_A + C_B ≠ const`）。
  - 修改律：`dC_A/dt = −k C_A^n`，`dC_B/dt = +η · k C_A^n`，`η ≠ 1`。
  - 量纲：两侧 `mol·m⁻³·s⁻¹` ✓。
  - 保留：T-trans，Arrhenius（`k` 仍 Arrhenius），正定性，稀释 scale 在 `n=1` 极限，dimensional homogeneity。

  - 安全范围：`C_A` 单调减到 0；`C_B` 单调到 `η · C_A(0) / 1` 上界（`η < 1.5` ⇒ 有界）。
  - 文献角度：借 branching ratio 概念；作为 system-level hidden-channel 总损耗、`η` 跨 1 双侧随机 ⇒ counterfactual。与 Part A δ-5-1（Q 泄漏）是同源 "守恒律 leakage" motif 的化学 instantiation。

## Domain 12：放射性衰变

**Baseline law**：`dN/dt = −λ N`。量纲：`[N] = count (or mol), [λ] = s⁻¹`。
**Invariants**：时间平移、Markov 无记忆性、线性 `N → a N`、粒子数守恒（链 `A → B`：`N_A + N_B = const`）、外场无关性（标准核衰变 `λ` 对常规外场免疫）、统计独立性（每个核独立）。

### Tier-1 (γ)

- **shift-γ-12-1 —— 密度耦合衰变率（受激辐射 motif 的跨域重皮）**
  - 动机：制造"受激增减"型核衰变 —— 真核衰变本质线性 / 单粒子；该非线性即 counterfactual。借激光受激辐射 `dN/dt ∝ N²` 动机但不复制 Einstein A/B 系数形式。
  - 破坏：线性 `N → aN`（出现内禀 `N_0` 标度）。
  - 修改律：`dN/dt = −λ N · (1 + α (N / N_0)^p)`。
  - 量纲：`[s⁻¹]` ✓（括号无量纲）。
  - 保留：T-trans，Markov（仍只依赖当前 `N`），粒子守恒（`A → B` 链同步该非线性 `dN_B/dt = +λ N_A (1 + α(N_A/N_0)^p)`），外场无关，统计独立性（mean-field 近似下）。
  - 采样：`λ ~ LogUniform(1e-6, 1e-1) s⁻¹`；`α ~ Uniform(-0.4, 0.8)`；`p ~ Uniform(0.3, 1.5)`；`N_0 ~ LogUniform(1e3, 1e8)`。
  - 安全范围：`α > -1` 保 `λ_eff > 0`；高 `N` 时 `dN/dt ∝ N^{1+p}`（α>0）超线性但 `N` 单调减 ⇒ 不发散。
  - 文献角度：借受激辐射 motif；搬到本质线性的核衰变 ⇒ counterfactual。

- **shift-γ-12-2 —— 参数调制衰变率（Part A δ-4-1 / δ-6-1 motif 的跨域重皮）** *(round-1 修复 B6 —— 锁相 0 以保 T-rev)*
  - 动机：外场调制衰变率 —— 真核衰变几乎免疫外场，故是清晰 counterfactual。借 Part A pendulum / RLC 的参数泵入 motif。原版随机 `φ` 被 invariant-checker 指出会同时破 T-rev ⇒ dual break；**锁 `φ ≡ 0`** 使 `cos(ωt)` 在 `t=0` 偶 ⇒ 仅破 T-trans。
  - 破坏：时间平移。
  - 修改律：`dN/dt = −λ(t) N`，`λ(t) = λ_0 [1 + ε cos(ω t)]`。
  - 量纲：`[s⁻¹]` ✓。
  - 保留：线性 `N → aN`（固定 `t` 下），Markov，粒子守恒（链 `A → B` 共享同一 `λ(t)`），T-rev ✓（`cos(ωt)` 在 `t→-t` 不变）。"外场无关性"是 T-trans 的衍生不变量，按 Part A 惯例算一条 (T-trans) 被破。
  - 采样：`λ_0 ~ LogUniform(1e-6, 1e-1)`；`ε ~ Uniform(0.05, 0.40)`；`ω ~ LogUniform(1e-3, 1) s⁻¹`。仿真 t 窗口建议中心化在 `t=0` (`t ∈ [-T/2, T/2]`) 以保 T-rev 在数值层可观测。
  - 安全范围：`ε < 0.5 ⇒ λ(t) > 0 ∀t`；`ω` 跨多 decade 给 AI 识别难度。
  - 文献角度：借参数共振 / Floquet motif；应用于核衰变 ⇒ counterfactual。与 Part A δ-4-1 / δ-6-1 同款 `φ=0` 约定。

### Tier-2 (δ)

- **shift-δ-12-1 —— 暗通道分支损耗**

## 必修项 —— 给 physicist-A（Part A）

**A1. γ-1-2（Hooke 2D 各向异性，ROT 破缺）。** `F = −K(θ) r r̂`，其中 `K(θ) = k₀[1+ξ cos(2(θ−φ))]`，是一个幅度依赖角度的纯径向力。所声称的势能 `V = ½K(θ)r²` 实际同时具有径向**和**切向梯度分量 —— 故所给的力并非 `−∇V`，系统非保守 ⇒ 能量守恒**也**被破 ⇒ 双对称性破缺，违反 D6 规则 §5。
- **修复**：从势能反推力，`F = −∇V` with `V = ½K(θ)r²`，得 `F_r = −K(θ)r`，`F_θ = −(r/2)·dK/dθ`。切向分量正是 `L_z` 不守恒的来源 —— 此时 ROT 是唯一被破对称性，E 与 T-rev 安全。

**A2. γ-2-1（引力四极矩，ROT 破缺）。** 与 A1 同样的结构性错误：`F = −G_eff(r̂;n̂)·m₁m₂ r̂/r²` 是幅度依角度变化的纯径向力，故非保守 ⇒ E 也被破。
- **修复**：同方剂 —— 从 `V(r,θ) = −G₀ m₁ m₂ · [1 + ξ((r̂·n̂)² − ⅓)] / r`（或等价式）出发，力 = `−∇V`。此时 ROT 为唯一破缺。

**A3. γ-4-1（摆，声称 PAR 破缺）。** 函数 `sin(θ − α tanh(θ/θ₀))` **关于 θ 为奇函数**（tanh 为奇，自变量关于 θ 为奇，奇自变量的 sin 仍为奇）。在 θ→−θ 下整条 EOM 不变 ⇒ PAR *被保留*，该 shift 并未破坏其所声称的对称性。
- **修复**：向 EOM 加入真正关于 θ 为偶的项。例如 `θ̈ + (g/L)·sin θ + (g/L)·α·(1−cos θ) = 0` —— `(1−cos θ)` 为偶，故 EOM 不再 PAR 不变。θ=0 处的平衡仍保留（因为 1−cos 0 = 0），小角展开多出非对称的 `αθ²/2` 立方型势能修正。采样 `α ~ Uniform(0.05, 0.5)` 可保平衡稳定。

**A4. γ-5-1（Coulomb 各向异性，ROT 破缺）。** 与 A1/A2 同样的非保守径向力问题。
- **修复**：同方剂 —— 定义 `V_pair(r,θ) = (k q_i q_j / r)·[1 + χ((r̂·m̂)² − ⅓)]`，并取 `F = −∇V_pair`。两体可加性保留；ROT 单破。

**A5. γ-5-2（Coulomb 饱和场，LIN 破缺）。** `E_eff(x) = f(|E_lin|) · E_lin(x)` 一般具有非零旋度，因为 `∇f` 不平行于 `E_lin`（在多源几何中 `|E_lin|` 的等位面与 `E_lin` 不垂直）。故 `E_eff` 无法写成 `−∇φ_eff` ⇒ 能量不守恒 ⇒ 双破缺。
- **修复**：把非线性放到*势能*而非场上。例如定义 `φ_eff = φ_lin + ξ · φ_lin³ / (φ_lin² + φ₀²)`（或类似的可饱和标量变换），再取 `E_eff = −∇φ_eff`。这样自动保守；LIN/叠加原理仍被破。

**A6. δ-2-1（引力 G(t) 调制，T-trans 破缺）。** 采样 `φ ~ Uniform(0, 2π)` 使 `sin(ωt+φ)` 在时间上一般不为偶 ⇒ 在 T-trans 之外 T-rev 也被破，构成双破缺。目录里"选择对称 φ"的回避说法在随机 φ 下不可实施。
- **修复**：固定 `φ ≡ 0` 并改为 `G(t) = G₀[1 + β cos(ω_G t)]`（关于 t=0 偶）。修复风格与 δ-4-1 完全一致，后者已正确采用此形式。

**A7. γ-6-2（RLC 非互易互感，Onsager 破缺）—— 仅文档备注。** 非对称的 `M₁₂ ≠ M₂₁` 使能量 `½ΣLᵢiᵢ² + M·i₁i₂` 不严格守恒（残留 `(M₁₂−M₂₁)·(i₁ di₂/dt − i₂ di₁/dt)` 项）。这是与 T-rev 破缺伴生的 Noether 配对 E 损失；结构上与 δ-1-1 / δ-3-1 的捆绑一致。

## 必修项 —— 给 physicist-B（Part B）

**B1. γ-7-1（热各向异性传导率，SO(3) 破缺）。** `tanh(|∇T|/G₀)` 门控引入了依赖场幅的非线性。在基线抛物自相似标度 `(t→λt, x→λ^{1/2}x, T→T)` 下，`|∇T|→λ^{−1/2}|∇T|` ⇒ tanh 自变量变化 ⇒ 抛物尺度**独立地**被破，与 SO(3) 同时破缺。双破缺。

**B4. γ-10-1（流体各向异性 KE，SO(3) 破缺）。** 仅修改 Bernoulli 标量 `½ vᵢ Mᵢⱼ vⱼ` 而未指定相应的 Euler / 动量方程修改是不完整的 —— Bernoulli 是 Euler 沿流线的积分，二者须自洽。
